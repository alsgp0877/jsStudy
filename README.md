```
> js 구성요소
- 객체 숫자 문자 불린(Boolean) null undefined
-- 객체(함수,배열)
> js 객체 표현법
  var 제로초 = {
    이름 : '제로초', ---속성(프로퍼티)
    먹다 : function 먹다(){ console.log("냠냠") } -- 속성(메서드)
  }
> 1. 객체
>> 객체 접근방법
- 제로처['이름']
+ 객체 접근 방법인데 객체가 아닌 문자열에서도 그 쓰임이 가능하다 ex) '안녕하세요'[3] -> 세
- 제로초.이름
+ '안녕하세요'.length도 문자열인데 객체처럼 쓰임이 가능하다 -- 나중에 설명
-- 그럼 두 접근 법의 차이는 무엇인가? '[]'안에는 변수를 쓸 수 있고 '.'는 변수를 쓸수 없다. 변수를 쓰게 되면 위치접근이 세세하게 가능하다.

> 2. 배열 : 각 그룹화는 하고 싶지만 속성 이름을 따로 주고 싶지 않을 때
  var 배열 = ['사과','오렌지','포도']
  var 배열같은객체 = { 0:'사과', 1:'오렌지', 2:'포도'}
* 배열 같은 객체 구별법 Array.isArray(배열)
+ 객체 선언을 해주지 않고도 쓸 수 있는 이유는 브라우저에서 제공해주기 때문에 ex) Math, Array, Number window document ..첫글자 대문자인건 생성자 ..


> 구구단 구현하기
> Window.prompt() : 사용자가 텍스트를 입력할 수 있도록 안내하는 메시지를 가지고 있는 대화상자를 설치한다.
- result = window.prompt(message,default);
- result는 사용자로 부터 입력받은 값이 들어간다.
- 모든 입력 데이터를 문자 정보로 인식한다. Number() 함수로 문자 자료형을 숫자 자료형으로 바꿔줘야한다. 그런 후 연산

> Math.random()은 0~1 사이의 숫자를 랜더함게 출력해준다 소수점 몇자리 아래까지 출력해준다.
- x 9 를 해주면 0~9 사이의 숫자로 뽑아준다.
- Math.floor(Math.random()) 은 내림을 해서 소숫점을 없애줌 + 올림 Math.ceil
```
